## Who built it

I'm Garrett Elias, a product manager working in Canadian agtech. I built this as a solo project to fill a gap I kept running into and to demonstrate what a non-technical PM with AI-enabled execution skills can ship independently.

The project was built with significant AI assistance -- Claude handled the bulk of the code generation, data modeling, and documentation drafting across Claude Code (agentic coding) and Claude.ai (planning and content). My role was product direction, decision-making, verification, and the industry knowledge that made the output trustworthy. The grain data was (painstakingly) manually verified against CGC source pages before being loaded into the API.

The project is open source. You can review the code at [github.com/garrettelias/gradingfactors-api](https://github.com/garrettelias/gradingfactors-api).
